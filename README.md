### 6-課題概要

本課題では，A~Eおよび〇，×の文字をホワイトボード(200mm×300mm)上に描画する「お絵描きロボット」を開発する．\
以下のレベル別タスクを段階的に実装する

- **レベル1**: 〇および×を描画できる
- **レベル2**: ローマ字を1文字描画できる
- **レベル3**: 複数の文字を連続して描画できる (ex: A→C→×→E)
- **レベル4**: シリアル通信により指定された1文字を描画できる
- **レベル5**: シリアル通信により指定された文字列を連続して描画できる (ex: B→C→A→E)

### 7-順運動学と逆運動学の実装

#### 順運動学

図1：手先延長型5節リンク機構の幾何学モデル

1. 各関節のグローバル座標定義

まず、固定端である $J_1$ を原点 $(0,0)$ とします。このとき、各関節の座標は以下のように定義されます。

$J_2$ の座標

$J_2$ は $J_1$ から $x$ 軸方向に距離 $l_1$ だけ離れているため：


$$x_2 = l_1, \quad y_2 = 0$$

$J_3$ の座標

リンク $l_2$ の回転角 $\theta_1$ を用いると：


$$x_3 = l_2 \cos \theta_1, \quad y_3 = l_2 \sin \theta_1$$

$J_4$ の座標

$J_2$ を起点とするリンク $l_3$ の回転角 $\theta_2$ より：


$$x_4 = l_1 + l_3 \cos \theta_2, \quad y_4 = l_3 \sin \theta_2$$

2. ローカル座標系における解析

三角形 $J_3 J_4 J_5$ に注目します。$J_3$ を原点とし、$J_3 \to J_4$ の方向を $x'$ 軸とするローカル座標系を定義します。

リンク間距離 $d_{34}$

$$d_{34} = \sqrt{(x_4 - x_3)^2 + (y_4 - y_3)^2}$$

閉回路方程式（複素数表現）

リンク機構は閉ループを形成するため、以下のように表されます。


$$d_{34} e^{i\theta_a} + l_5 e^{i\theta_b} = l_4 e^{i\theta_c}$$

絶対値を用いた幾何拘束

右辺を $x_5 + i y_5$ とおくと、


$$l_5 e^{i\theta_b} = (x_5 - d_{34}) + i y_5$$


両辺の絶対値を取ることで、以下の幾何拘束が得られます。


$$(x_5 - d_{34})^2 + y_5^2 = l_5^2$$

円の方程式（リンク $l_4$）

また、$J_5$ は $J_3$ を中心とした半径 $l_4$ の円弧上にあるため：


$$x_5^2 + y_5^2 = l_4^2$$

連立による $J_5$ のローカル座標

上記2式を連立して解くことで、ローカル座標 $(x'_5, y'_5)$ を得ます。


$$x'_5 = \frac{d_{34}^2 + l_4^2 - l_5^2}{2 d_{34}}$$

$$y'_5 = \sqrt{l_4^2 - (x'_5)^2}$$

ペン先 $J_6$ のローカル座標

$J_6$ はリンク $l_4$ の延長上にあるため、幾何学的な相似比（$\frac{l_4 + l_6}{l_4}$）より以下のように導出されます。

$$x'_6 = \frac{l_4 + l_6}{l_4} \cdot x'_5 = \frac{l_4 + l_6}{l_4} \cdot \frac{d_{34}^2 + l_4^2 - l_5^2}{2 d_{34}}$$

$$y'_6 = \frac{l_4 + l_6}{l_4} \cdot y'_5 = \frac{l_4 + l_6}{l_4} \cdot \sqrt{l_4^2 - (x'_5)^2}$$

3. グローバル座標系への変換

ローカル座標を回転行列によってグローバル座標（原点 $J_1$）へ変換します。

回転角 $\alpha$

$$\alpha = \operatorname{atan2}(y_4 - y_3,\; x_4 - x_3)$$

最終的なペン先座標 $(x_6, y_6)$

$$x_6 = x_3 + x'_6 \cos \alpha - y'_6 \sin \alpha$$

$$y_6 = y_3 + x'_6 \sin \alpha + y'_6 \cos \alpha$$
