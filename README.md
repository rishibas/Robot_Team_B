### 6-課題概要

本課題では，A~Eおよび〇，×の文字をホワイトボード(200mm×300mm)上に描画する「お絵描きロボット」を開発する．\
以下のレベル別タスクを段階的に実装する

- **レベル1**: 〇および×を描画できる
- **レベル2**: ローマ字を1文字描画できる
- **レベル3**: 複数の文字を連続して描画できる (ex: A→C→×→E)
- **レベル4**: シリアル通信により指定された1文字を描画できる
- **レベル5**: シリアル通信により指定された文字列を連続して描画できる (ex: B→C→A→E)

### 7-順運動学と逆運動学の実装

#### 順運動学

<img width="420" height="300" alt="Image" src="https://github.com/user-attachments/assets/6eb0ded0-7139-4f37-a713-aa4351397d0e" />

図1：手先延長型5節リンク機構の幾何学モデル


1. 各関節のグローバル座標定義

まず、$J_1$ を原点 $(0, 0)$ とし、各関節の座標を以下のように定義します。

$J_2$ の座標:
$J_2$ は $J_1$ から $x$ 軸方向に距離 $l_1$ だけ離れているため：


$$x_2 = l_1, \quad y_2 = 0$$

$J_3$ の座標:
リンク $l_2$ の回転角 $\theta_1$ を用いて：


$$x_3 = l_2 \cos \theta_1, \quad y_3 = l_2 \sin \theta_1$$

$J_4$ の座標:
$J_2$ を起点としたリンク $l_3$ の回転角 $\theta_2$ を用いて：


$$x_4 = l_1 + l_3 \cos \theta_2, \quad y_4 = l_3 \sin \theta_2$$

2. ローカル座標系における解析

$J_3, J_4, J_5$ で構成される三角形に注目し、$J_3$ を原点、$J_3 \to J_4$ の方向を $x'$ 軸とするローカル座標系を定義します。

$J_3$ と $J_4$ の距離を $d_{34}$ とすると：


$$d_{34} = \sqrt{(x_4 - x_3)^2 + (y_4 - y_3)^2}$$

$J_5$ のローカル座標 $(x'_5, y'_5)$

$J_3$ を中心とする半径 $l_4$ の円と、$J_4$ を中心とする半径 $l_5$ の円の交点として求められます。


$$x'_5 = \frac{d_{34}^2 + l_4^2 - l_5^2}{2 d_{34}}$$

$$y'_5 = \sqrt{l_4^2 - (x'_5)^2}$$

ペン先 $J_6$ のローカル座標 $(x'_6, y'_6)$

ペン先 $J_6$ はリンク $l_4$ の延長線上に位置するため、相似比（$\frac{l_4 + l_6}{l_4}$）を用いて以下のように表せる．

$$x'_6 = \frac{l_4 + l_6}{l_4} \cdot x'_5 = \frac{l_4 + l_6}{l_4} \cdot \frac{d_{34}^2 + l_4^2 - l_5^2}{2 d_{34}}$$

$$y'_6 = \frac{l_4 + l_6}{l_4} \cdot y'_5 = \frac{l_4 + l_6}{l_4} \cdot \sqrt{l_4^2 - (x'_5)^2}$$

3. グローバル座標系への変換

算出したローカル座標を、回転行列を用いてグローバル座標系（$J_1$ 原点）へ変換します。
$J_3$ から $J_4$ への傾き角を $\alpha$ とします。

$$\alpha = \operatorname{atan2}(y_4 - y_3, x_4 - x_3)$$

最終的なペン先 $J_6$ の座標 $(x_6, y_6)$ は以下の通りです。

$$x_6 = x_3 + x'_6 \cos \alpha - y'_6 \sin \alpha$$

$$y_6 = y_3 + x'_6 \sin \alpha + y'_6 \cos \alpha$$

この一連の計算により、サーボモータの指令角からリアルタイムにペン先の位置を特定できる．
