### 6-課題概要

本課題では，A~Eおよび〇，×の文字をホワイトボード(200mm×300mm)上に描画する「お絵描きロボット」を開発する．\
以下のレベル別タスクを段階的に実装する

- **レベル1**: 〇および×を描画できる
- **レベル2**: ローマ字を1文字描画できる
- **レベル3**: 複数の文字を連続して描画できる (ex: A→C→×→E)
- **レベル4**: シリアル通信により指定された1文字を描画できる
- **レベル5**: シリアル通信により指定された文字列を連続して描画できる (ex: B→C→A→E)

#### 順運動学

1. 各関節のグローバル座標定義

まず、固定端である $J_1$ を原点 $(0,0)$ とする．このとき、各関節の座標は以下のように定義される．

$J_2$ の座標

$J_2$ は $J_1$ から $x$ 軸方向に距離 $l_1$ だけ離れているため：


$$x_2 = l_1, \quad y_2 = 0$$

$J_3$ の座標

リンク $l_2$ の回転角 $\theta_1$ を用いると：


$$x_3 = l_2 \cos \theta_1, \quad y_3 = l_2 \sin \theta_1$$

$J_4$ の座標

$J_2$ を起点とするリンク $l_3$ の回転角 $\theta_2$ より：


$$x_4 = l_1 + l_3 \cos \theta_2, \quad y_4 = l_3 \sin \theta_2$$

2. ローカル座標系における解析

三角形 $J_3 J_4 J_5$ に注目します。$J_3$ を原点とし、$J_3 \to J_4$ の方向を $x'$ 軸とするローカル座標系を定義する．

リンク間距離 $d_{34}$

$$d_{34} = \sqrt{(x_4 - x_3)^2 + (y_4 - y_3)^2}$$

閉回路方程式（複素数表現）

リンク機構は閉ループを形成するため、以下のように表せる。


$$d_{34} e^{i\theta_a} + l_5 e^{i\theta_b} = l_4 e^{i\theta_c}$$

絶対値を用いた幾何拘束

右辺を $x_5 + i y_5$ とおくと、


$$l_5 e^{i\theta_b} = (x_5 - d_{34}) + i y_5$$


両辺の絶対値を取ることで、以下の幾何拘束が得られる．


$$(x_5 - d_{34})^2 + y_5^2 = l_5^2$$

円の方程式（リンク $l_4$）

また、$J_5$ は $J_3$ を中心とした半径 $l_4$ の円弧上にあるため：


$$x_5^2 + y_5^2 = l_4^2$$

連立による $J_5$ のローカル座標

上記2式を連立して解くことで、ローカル座標 $(x'_5, y'_5)$ を得る．


$$x'_5 = \frac{d_{34}^2 + l_4^2 - l_5^2}{2 d_{34}}$$

$$y'_5 = \sqrt{l_4^2 - (x'_5)^2}$$

ペン先 $J_6$ のローカル座標

$J_6$ はリンク $l_4$ の延長上にあるため、幾何学的な相似比（$\frac{l_4 + l_6}{l_4}$）より以下のように導出する．

$$x'_6 = \frac{l_4 + l_6}{l_4} \cdot x'_5$$

$$y'_6 = \frac{l_4 + l_6}{l_4} \cdot y'_5$$

3. グローバル座標系への変換

ローカル座標を回転行列によってグローバル座標（原点 $J_1$）へ変換する．

回転角 $\alpha$

$$\alpha = atan2(y_4 - y_3,\ x_4 - x_3)$$

最終的なペン先座標 $(x_6, y_6)$

$$x_6 = x_3 + x'_6 \cos \alpha - y'_6 \sin \alpha$$

$$y_6 = y_3 + x'_6 \sin \alpha + y'_6 \cos \alpha$$

<br>

#### 逆運動学

1. 左側リンク系による $\theta_1$ の導出

リンク $l_2$ の先端 $J_3$ と、ペン先 $P$ を結ぶ直線長は $l_4 + l_6$ で固定されている。これに着目し、原点 $J_1(0, 0)$ を中心とする円と、$P(x, y)$ を中心とする円の交点として $J_3$ を求める。

幾何拘束方程式

$J_3(x_3, y_3)$ は以下の二つの条件を満たす必要がある。

リンク $l_2$ の制約:


$$x_3^2 + y_3^2 = l_2^2$$

リンク $l_4 + l_6$ の制約:


$$(x - x_3)^2 + (y - y_3)^2 = (l_4 + l_6)^2$$

$\theta_1$ の解法（余弦定理の適用）

$J_1$ から $P$ までの距離を $d_{1P} = \sqrt{x^2 + y^2}$ とする。三角形 $J_1 J_3 P$ において、余弦定理より $\angle P J_1 J_3$ を $\alpha$ と定義する。

$$\cos \alpha = \frac{l_2^2 + d_{1P}^2 - (l_4 + l_6)^2}{2 \cdot l_2 \cdot d_{1P}}$$

$J_1$ から見た $P$ の方位角を $\phi_1 = \operatorname{atan2}(y, x)$ とすると、$\theta_1$ は以下のようになる。

$$\theta_1 = \phi_1 \pm \alpha$$


（※機構の折りたたみ方向に応じて符号を選択する）

2. 右側リンク系への展開

$\theta_1$ が決定することで、$J_3$ の座標が確定する。


$$x_3 = l_2 \cos \theta_1, \quad y_3 = l_2 \sin \theta_1$$

中間関節 $J_5$ の特定

前述の通り、$J_3, J_5, P$ は一直線上に位置するため、相似比を用いて $J_5$ を算出する。

$$\vec{u} = \frac{(x - x_3, y - y_3)}{l_4 + l_6}$$

$$J_5 = J_3 + l_4 \cdot \vec{u}$$

3. 右側リンク系による $\theta_2$ の導出

次に、固定点 $J_2(l_1, 0)$ から伸びるリンク $l_3$ と、確定した $J_5$ を結ぶリンク $l_5$ の幾何拘束を解く。

幾何拘束方程式

$J_4(x_4, y_4)$ は以下の二つの条件を満たす。

リンク $l_3$ の制約:


$$(x_4 - l_1)^2 + y_4^2 = l_3^2$$

リンク $l_5$ の制約:


$$(x_5 - x_4)^2 + (y_5 - y_4)^2 = l_5^2$$

$\theta_2$ の解法

$J_2$ から $J_5$ までの距離を $d_{25} = \sqrt{(x_5 - l_1)^2 + y_5^2}$ とする。三角形 $J_2 J_4 J_5$ において、$\angle J_5 J_2 J_4$ を $\beta$ と定義する。

$$\cos \beta = \frac{l_3^2 + d_{25}^2 - l_5^2}{2 \cdot l_3 \cdot d_{25}}$$

$J_2$ から見た $J_5$ の方位角を $\phi_2 = \operatorname{atan2}(y_5, x_5 - l_1)$ とすると、$\theta_2$ は以下のようになる。

$$\theta_2 = \phi_2 \pm \beta$$
